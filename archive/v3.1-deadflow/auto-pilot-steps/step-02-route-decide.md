# Step 02 — 策略执行

## MANDATORY EXECUTION RULES
- 🛑 守卫层命中时必须中断
- 📋 质量 ≥ 60 自动通过，< 60 中断用户

## 策略矩阵

### 🚀 直达模式（复杂度 1-3）
直接 Read/Edit/Bash 执行。无需调度 agent。结束后输出简短结果。

### 🎯 精准调度（复杂度 4-6）
1. 选最匹配 agent（烛自动选评审模式）
2. 静默准备任务卡
3. 召唤 agent → 读下游建议 → 链式调度（上限 3 层）
4. ≥ 60 综合输出 / < 60 重试

### 🔄 全管道（复杂度 7-10）
自动执行 workflow skill 的 6 阶段，采用自动推进模式：
- Phase 0 增强：静默，≥ 5 自动过
- Phase 1 调研：静默调研 + 自动综合
- Phase 2 规划：静默规划 + 落 handoff
- Phase 3 执行：自动执行，仅守卫层中断
- Phase 4 验证：≥ 60 自动过 / < 60 中断
- Phase 5 交付：输出最终结果

## 链式调度规则
每个 agent 返回后读 `## 下游建议`。有推荐 → 注入前序发现 → 链式召唤。
维护 `accumulated_findings`。链式上限 3 层。

| ⛔ 质量 < 60 | 选方向 |
| ❓ 意图不清 | 补信息 |
| 🔀 等价方案 | 选方案 |
