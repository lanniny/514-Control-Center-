# Step 01 — 澄清式提问

## MANDATORY EXECUTION RULES
- 🛑 字段不全时绝不出规格
- 📋 你是引导者——引导用户思考，不是替用户决定

## 7 字段澄清

| 字段 | 缺则必问 |
|------|---------|
| Who（受益人）| "给谁用？" |
| What（具体行为）| "最小可用版的典型场景？" |
| Why（动机）| "现在怎么解决？为什么不够？" |
| Done（完成标准）| "做到什么程度算完成？" |
| Out-of-scope | "这一版不做什么？" |
| Constraint | "时间/工具/平台约束？" |
| Risk Appetite | "要稳还是要快？" |

**例外**：主人说"先出草稿" → 允许默认假设，但**明确列出所有假设**。

7 字段齐全后 → step-02。
